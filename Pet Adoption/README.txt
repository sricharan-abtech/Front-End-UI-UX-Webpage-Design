Pet Adoption Center
===================

Course: Front End UI/UX
Institution: Christ University

--------------------------------------------------
Project Overview
--------------------------------------------------
This is a responsive, single-page Pet Adoption Center website created using HTML5 and CSS3.
It showcases available pets, explains the adoption process, highlights success stories, and
provides a volunteer form for community engagement. The design focuses on a warm and
welcoming color palette, semantic HTML structure, and responsive layout using Flexbox.

Technologies Used
--------------------------------------------------
HTML5 – Semantic content structure
CSS3 – Layout and styling (Flexbox)
Placeholder images from PlaceKitten & PlaceDog

--------------------------------------------------
Features
--------------------------------------------------
- Responsive design for mobile, tablet, and desktop
- Available pets displayed in attractive card format
- Step-by-step adoption process guide
- Success stories with descriptive text
- Volunteer form with required fields

How to Run
--------------------------------------------------
1. Download or clone the repository.
2. Open index.html in a modern browser.
3. Ensure images folder is in the same directory as index.html for proper display.
