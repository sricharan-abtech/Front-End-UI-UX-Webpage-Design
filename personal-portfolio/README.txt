Personal Portfolio Website
==========================

Course: Front End UI/UX
Institution: Christ University

--------------------------------------------------
Project Overview
--------------------------------------------------
This is a static personal portfolio website created using only HTML5 and CSS3.  
It showcases academic and creative achievements, skills, projects, and contact details.  
The focus is on clean UI, semantic structure, responsive layout, and visual hierarchy.
